## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "ohm"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ohm
```

## Development

Install development dependencies:
```bash
$ bundle
```

Setup database:
```bash
$ bin/rails db:setup
$ bin/rails db:migrate
```
